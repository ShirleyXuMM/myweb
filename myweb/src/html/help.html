    新手指南

    新用户注册 :

    第一步 : 点击页面右上方的“ 注册 ”按钮注册 快乐购 账号。

    第二步 : 注册前请仔细阅读快乐购《用户服务协议》，如无异议请勾选“ 注册即表示同意快乐购 《用户服务协议》 条款 ”。通过手机号号码进行注册，请根据相应提示在信息栏内填入您的注册信息。


    登录 :

    第一步 : 如您已经拥有 快乐购 账号，请点击页面右上方的“ 登录 ”按钮 。

    第二步 :   在登录页面的信息栏内填入对应信息（用户名/邮箱/手机号），点击“登录”按钮进行登录，或通过选择登录框下方的合作账号进行快速登录。登录成功后，系统将自动跳转至 快乐购 首页。


    挑选商品 :

    方法一：您可以点击快乐购页面上方的 首页 、 T V直播、   全球购、 品牌馆，来查看每日最新的商品信息，挑选和购买您心仪的商品。

    方法二： 快乐购 经常举办促销、专场、抽奖等超值回馈活动，想要以最低的价格买到最优质的商品，就天天都来 快乐购 关注首页更新吧！

    方法三：您还可以在搜索栏输入关键字或者品牌名称 ， 快速找到商品 ； 按照商品分类找到您心仪的商品 ； 详情页图片下方点击收藏商品。

    购物车 :

    第一步 按照页面中的提示文字详细填写/核对配送地址，支付方式，商品清单等。相关信息核对无误后点击“ 提交 订单”前往支付页面，或  “返回购物车”进行修改。已获得 优惠 券或者快乐币的用户可同时选择使用优惠券和快乐币 。

    第二步 : 选择货到付款的用户，成功提交订单后将会直接显示订购成功，此时下单过程已完成，您只需等候快递员为您送货上门，或继续逛逛。选择在线支付的用户，成功提交订单后，请仔细阅读后点击“立即付款”进行支付，完成下单的最后一步。

    第三步 : 付款成功后将显示支付成功页面，您可以随时在“我的订单”中查看及操作您的订单了。


    货到付款订单 :

    请您在付款给送货上门的快递人员之前，先当面拆开包装箱，务必对照箱内发货单仔细核对，确认商品有无数量、质量等问题。确认无误之后再进行付款，否则可选择拒收并直接退回给快递人员。


    在线支付订单 :
